# Context Pack

**Tags:** infrastructure, continuity, session

**Status:** reviewed

## Summary
A Context Pack is the **session constitution**: it defines scope, constraints, canon pointers, and the working set so fresh chats behave consistently.

## Mechanics
Load the pack at session start; scope retrieval and decision-making by it. Continuity comes from structured context, not assumed memory.

## Non-goals
- This node is not canon by itself. Canon lives in `canon/decisions/*`.
- Do not treat chat summaries as binding without ratification.
