# Boot Pack

**Tags:** infrastructure, continuity

**Status:** reviewed

## Summary
A Boot Pack is a portable starter bundle that makes a fresh session instantly aligned: principles, canon snapshot, open loops, and an active context pack.

## Mechanics
Boot Packs should remain minimal and stable; they point to evidence and artifacts rather than embedding raw transcripts.

## Non-goals
- This node is not canon by itself. Canon lives in `canon/decisions/*`.
- Do not treat chat summaries as binding without ratification.
